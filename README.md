## Шаблоны для нахождения нуля функции

### Метод Ньютона

Представлены два вида функций:  
первый приближённо считает производную, основываясь на eps;  
второй принимает производную в виде аргумента.  

`findNullN_func` - ищет ноль в окрестности точки,  
`findsNullN_func` - ищет нули в интервалах, если:  
ValueNull - корень не был найден за представленное кол-во итераций;  
ValueNotRange - корень найден вне заданного интервала,  
`findsNullNsort_func` - ищет нули в интервалах (сортирует самостоятельно), если:  
ValueMissing - корень не был найден на отрезке, меняющем знак.  

Примеры: [examples/function/findN.cpp]

### Метод половинного деления

Вычисляют только на отрезках с разными знаками на концах.

`findNullL_func` - ищет ноль на интервале,   
`findsNullLsort_func` - ищет нули на интервалах (сортирует самостоятельно).  
Примеры: [examples/function/findL.cpp]

## Вспомогательные элементы

## Концепты

`numberRConcept` - концепт действительных чисел,  
`funcNumberRC` - концепт однозначных функций от действительных чисел,  
`vecPointsC` - концепт типа vector со вставкой в конец.  

## Шаблоны

`linspace` - разбивает отрезок на n равных равноудалённых точек, также вырожденные случаи при n = 0, n = 1 (левое значение),  
`transform_this` - преобразует значения внутри std::vector,  
`del_dublicate` - удаляет дубликаты с заданной точностью, можно добавить преобразование до удаления.  
Примеры: [examples/function/linspace.cpp]

## Классы

### Символические

`Succ` - "приближенный корень",  
`ValueMissing` - "потерянный корень",  
`ValueNotRange` - "вне диапазона корень",  
`ValueNull` - "числа не существует" (не выполнено достаточное условие Больцано — Коши).  

### Стандартные

`sqrtValue` - класс корня:  
`"<" ">" ">=" "<=" "!=" "=="` - операторы сравнения,  
`operator bool` - проверка на существование значения (отличное от std::nullopt),  
`operator*` - оператор разыменования, возвращает значение при условии "Succ", иначе выкидывает исключения,  
`get_code_error()` - возвращает коды ошибок (Succ, ValueNull, ValueMissing, ValueNotRange <=> 0, 1, 2, 3),  
`get_try_exception()` - выбрасывает исключения (ValueNull, ValueMissing, ValueNotRange),  
`get_range` - получить диапазон поиска корня,  
`set_range` - задаёт диапазон поиска корня (используется во внутренней логике),  
`get_value_not_valid` - получить внутреннее значение (std::optional),  
`get_value_not_optional_valid` - получить внутреннее значение,  
`set_exception(...)` - установить "тип корня" (используется во внутренней логике).  

### Примечание

Все представленные шаблоны имеют свою область применения, в частности, на рассматриваемом отрезке функция **должна** менять знак, в противном случае данный отрезок не рассматривается.  
Вначале стоит рассматривать производные, чтобы найти точки, подозрительные на экстремумы, и рассматривать их отдельно.
